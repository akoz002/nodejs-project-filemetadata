API Project: File Metadata Microservice

User Stories:

  1. I can submit a form object that includes a file upload.
  2. The form file input field has the "name" attribute set to "upfile".
  3. When I submit a file, I will receive the file name, type and size in bytes within the JSON response. For example:
  • {
    • "name": "Desert.jpg",
    • "type": "image/jpeg",
    • "size": 845941
  • }
